This is a login application that also displays the latest EUR/USD exchange rate and timestamp using a WebSocket feed.

## Features

- **Login Form:** User can log in with email and password.
- **Password Reset:** Modal popup to reset the password.
- **Real-Time Exchange Rate:** Display the latest EUR/USD exchange rate and timestamp.
- **WebSocket Integration:** Real-time data feed using WebSocket.

## Technologies Used

- **React:** A JavaScript library for building user interfaces.

- **React createPortal:** For rendering the modal component outside the main DOM hierarchy.

- **WebSocket:** For real-time data streaming.

- **SCSS:** For styling the components.

- **BEM Methodology:** Block Element Modifier naming convention for writing clean and manageable CSS.

## Getting Started

- **Clone the repository:** `git clone https://github.com/your-username/login-page.git`

- **Install:** `cd login-app` then `npm install`

- **Running:** run `npm start`

- **Testing:** run `npm test`

- **Building:** run `npm build`
